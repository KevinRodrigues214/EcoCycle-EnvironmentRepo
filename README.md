## üåç Overview

**EcoCycle Environment** is a mobile and web application that empowers users to:
- Locate nearby recycling stations  
- Track their recycling progress  
- Participate in local eco-friendly events  
- Earn **eco-points and cashback rewards**

The app leverages **AI object recognition** to identify recyclable materials and integrates **Google Maps** to visualize local recycling stations and events.  
It also provides **real-time recycling statistics** for each province, helping both citizens and governments monitor environmental progress.
